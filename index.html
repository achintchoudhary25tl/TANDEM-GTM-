Problem 01
BIM dies at handover
The richest asset dataset ever created — the BIM model — is abandoned the moment construction ends. Facility teams receive PDFs and spreadsheets instead.
🔌
Problem 02
BMS is operationally blind
BMS knows sensor states but not asset context. It cannot answer: "Which AHU serves Floor 3?" or "What is the maintenance history of this chiller?" without manual cross-referencing.
📊
Problem 03
Asset data lives in Excel
Asset registers, warranty documents, inspection schedules — all in spreadsheets, owned by individuals, not connected to live operational systems. One resignation = data loss.
The cost of this fragmentation is enormous: reactive maintenance, energy inefficiency, compliance risk, and an inability to benchmark performance across portfolios.

Ecosystem

Tier 1 — Specifiers
MEP & BIM Consultants + PMCs
These firms write the specifications. If Tandem is in their standard spec library, it gets included in every project they touch.

Tier 2 — Deployers

SIs execute BMS, SCADA, and IoT deployments. Training them to include Tandem integration in their delivery scope creates a scalable deployment network.

SLIDE 08 — SALES STRATEGY

The BOQ
Insertion Play

The single most effective GTM lever in India's infrastructure market is BOQ specification at design stage. Once a line item is in a BOQ, it is budgeted, tendered, and procured — automatically.

"If Tandem is not specified in the BOQ, it will never be deployed. If it is, it will almost certainly be purchased."

The play: train MEP consultants and PMC firms to include a standardized "Digital Twin Platform" BOQ section in every new build over 50,000 sq ft. This creates a self-reinforcing adoption pipeline.

Sample BMS BOQ — Smart Building Project
# BOQ Item Status
1.1 DDC Controllers (VAV, AHU, FCU) Standard
1.2 Field Sensors (Temp, CO₂, Pressure) Standard
1.3 BMS Software / SCADA License Standard
1.4 System Integration (BACnet/Modbus) Standard
1.5 Networking Infrastructure Standard
1.6 Digital Twin Platform (Autodesk Tandem) ◆ Add Now
1.7 Asset Data Integration & BIM Sync ◆ Add Now
1.8 Operational Data Mapping & Commissioning ◆ Add Now
Lines 1.6–1.8 represent the Tandem insertion — incremental revenue with zero channel conflict, budgeted at project design stage.
